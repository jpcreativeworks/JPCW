Alvernon Donuts
Although this project was submitted as part of my SheCodes curriculum, I chose to go beyond the assignment’s generic guidelines and design a site for a real local business—Alvernon Donuts. The business didn’t have a website, and I saw an opportunity to create something useful while pushing myself to apply real-world thinking. I handled the layout, styling, and responsive development using HTML, CSS, and Bootstrap, focusing on user experience and visual clarity. This project not only helped me fulfill the SheCodes criteria but also allowed me to approach web development from a more authentic, community-centered perspective.

Time Globe
Time Globe was a focused JavaScript project designed to display live, location-specific time across multiple cities. I implemented both moment.js and moment-timezone to handle real-time updates and international time zones. My biggest challenge was integrating the dynamic time updates with user interaction, but solving it helped sharpen my understanding of asynchronous behavior in JavaScript and DOM manipulation. The sleek, responsive interface also reflects my dedication to clean, functional design.
